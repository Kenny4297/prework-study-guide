Git

How to add and commit to a branch

  • Never write anything on the main branch: create a separate branch called “feature”. When you are ready to push things up to the main branch:
  • git add .
  • git commit -m “text here”
  • git pull origin master
  • git push origin “feature branch name here”
  • Then, go to the repo on github. Look for something that says “Compare & pull request”. If you do not see this, On the left hand side of the page, under “Code”, switch to the feature branch. You will then hopefully see something that says “behind of commits” and ask for a pull request there. Create pull requests, type in a title, and continue on!

    • More git tips:

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
